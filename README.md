# Interpolasi-Lagrange-untuk-Pendekatan-Nilai-Fungsi
Project ini membahas implementasi metode Interpolasi Lagrange untuk melakukan pendekatan nilai fungsi pada titik tertentu berdasarkan data numerik yang tersedia.  Metode ini digunakan untuk membentuk polinom interpolasi dari beberapa titik data sehingga dapat digunakan untuk memperkirakan nilai fungsi yang belum diketahui.
